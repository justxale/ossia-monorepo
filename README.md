# Ossia - PET Project of Web-service for track listening and uploading
## Microservices
- Tracks + Creators API Service
- User Service
- Sync Service (via WebSockets)

## Used frameworks
- Nuxt 4 + Vite (Typescript, npm)
- FastAPI (Python, uv)

## Used infrastructure
- PostgreSQL
- RabbitMQ
- MinIO (And cloud-based S3-like object storages)
- nginx
- ffmpeg
- Docker Compose
